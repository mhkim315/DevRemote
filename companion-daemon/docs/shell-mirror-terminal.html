Shell Mirror Terminal

Sessions let you run multiple terminals simultaneously.

Creating

  • • Tap "Sessions" → "+ New Session"
  • • Each session runs independently

Switching

  • • Tap "Sessions" dropdown
  • • Select any session
  • • Your processes keep running in background

Limits

  • • Max 10 sessions per Mac
  • • Auto-deleted after 24h inactive

Connection stuck?

  • • Wait 10 seconds for auto-retry
  • • Still red? Return to Dashboard → Reconnect

Slow connection?

  • • App tries: Local network → WebRTC → Fallback
  • • First connect may take 30 seconds
  • • WebRTC works best (100-500ms)

Behind corporate firewall?

  • • May need IT to whitelist STUN servers
  • • Contact: stun.l.google.com port 19302
